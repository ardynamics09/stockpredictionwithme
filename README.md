An interactive stock prediction web application that fetches real market data and visualizes historical trends along with future price predictions.

## 🚀 Live Demo

https://ardynamics09.github.io/stockpredictionwithme/

## 📌 Features

* Real-time stock market data using **Alpha Vantage API**
* Interactive **stock price graph** using Chart.js
* **Future stock price prediction** based on historical growth
* **Risk–Return analysis**
* **Prediction confidence score**
* **Buy / Hold / Avoid investment signal**
* **Mobile responsive UI**
* Loading animation for better user experience

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript
* Chart.js
* Alpha Vantage API

## 📊 How It Works

1. User selects a company and number of shares.
2. The app fetches **historical stock data** using Alpha Vantage API.
3. Past **1 year stock prices** are visualized using an interactive graph.
4. Future price is predicted using a **growth-based model**.
5. The system calculates:

   * Profit/Loss percentage
   * Risk-Return ratio
   * Prediction confidence
   * Investment signal (Buy / Hold / Avoid)
